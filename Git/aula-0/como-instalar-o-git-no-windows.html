Nesse tutorial vamos explicar uma alternativa fácil para instalar o git no windows. O Git-for-Windows (git para windows) é um aplicativo leve que trás todas as ferramentas do git desenvolvido para linux para o ambiente windows. Para os usuários experientes, que já estão familiarizados com as linhas de comando do Git, ele consta com um emulador do Bash capaz de interpretar os comandos. Isso permite tanto uma interoperabilidade entre programas de diferentes sistemas operacionais como uma documentação unica. Para usuários do windows ele conta com interfaces gráficas e opções de navegação a partir do menu de contexto do Explorer. As interfaces gráficas permitem uma fácil assimilação das idéias de versionamento trazidas pelo Git.

Materiais necessário para o acompanhamento do Tutorial

Para esse tutorial será necessário:

  • Um computador com o sistema operacional Windows 7.
  • O download do software Git-For-windows.

Como instalar o Git para o windows

  • Passo 1: Clique no arquivo Git-for-windows baixado e o execute;
  • Passo 2: Na janela de informação da licença, figura 1a, clique no botão “next” para aceitar a licença GNU;
  • Passo 3: Na janela de seleção do local de instalação, figura 1b, clique no botão “Browse..” para selecionar o caminho que será instalado o git-for-windows e então clique no botão “next” para seguir a instalação;
  • Passo 4: Na janela de seleção dos componentes da instalação, figura 2a, deixe selecionado as opções “Windows Explorer Integration”, “Git Bash Here”, “Git Gui Here”, “Associata .git* configuration filies with the default text editor”, “Associate .sh files to be run with bash”. Após a seleção das opções pressione o botão “next” para seguir a instalação.
  • Passo 5: Na janela de criação da pasta do menu iniciar, figura 2b, clique no botão “next” para continuar”.
  • Passo 6: Na janela de ajuste das variáveis de ambiente, figura 3a, selecione a opção “Use Git form the Windows Commando Prompt” então clique no botão “next” para continuar a instalção.
  • Passo 7: Na janela de configuração da edição das linha de conversão, figura 3b, selecione a opção “Checkout Windows-stype, commit Unix-style line endings” então clique no botão “next” para continuar a instalação.

  • Passo 6: Na janela de configuração do emulador de terminal, figura 4a, selecione a opção “Use MinTTy (the default terminal of MSYS2” então clique no botão “next” para continuar a instalção.
  • Passo 7: Na janela de configurações extra, figura 4b,selecione a opção “Enable file system caching” então clique no botão “next” para continuar a instalção.
  • Passo 8: Na janela de Janela das opções experimentais, figura 4a, desmarque selecione a opção “Use MinTTy (the default terminal of MSYS2” então clique no botão “next” para continuar a instalação.
  • Passo 9: Na janela de Janela de instalação, figura 46, acompanhe a instalação do git-for-windows.
  • Passo 10: Na janela de finalização da instalção do Git for Windows clique em finish para fechar a janela de instalção.

Testando a instalação do Git-For-Windows

Clique com o botão direito e abra o menu de contexto do explorer depois verifique se está habilitado a função “Git Gui Here” e ” Git Bash Here”. Caso você encontre essas novas duas funçoes o Git-For-Windows foi instalado corretamente.

Pronto, está instalado o Git-for-windows. Agora você pode utilizar o git no windows. Mas para utilizar os comandos do Git de forma mais intuitiva instale o TortoiGit .
